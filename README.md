# YouTube
## 1. Тема и целевая аудитория
### MVP
1. Создание аккаунта (канала)
2. Возможность выкладывать видео
3. Просмотр видео
4. Оценка видео (лайк/дизлайк), комментарии под роликом
5. Подписка на канал
6. Поиск видео
7. Система рекомендаций, на основе просмотренных ранее роликов
